# FlowSpec Output

This directory contains machine-readable flow JSON files generated by BluePrints.

## Purpose

Store structured, programmatic representations of user flows and process diagrams for:
- Automated testing
- Integration with other tools
- Version tracking
- Analysis and reporting

## Format

JSON files in this directory follow a standardized schema that represents:
- Flow nodes and edges
- Decision points
- User actions
- System responses

Files will be generated automatically as flows are created.
